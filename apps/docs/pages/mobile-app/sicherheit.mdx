# Sicherheit

Die Sicherheitseinstellungen in der mobilen App umfassen Zwei-Faktor-Authentifizierung, Sitzungsverwaltung und Spracheinstellungen.

## 2FA einrichten

Die Zwei-Faktor-Authentifizierung (2FA) schützt Ihr Konto mit einem zusätzlichen Code bei der Anmeldung.

### Aktivierung

1. Öffnen Sie **Einstellungen → Sicherheit**.
2. Tippen Sie auf **2FA aktivieren**.
3. Scannen Sie den angezeigten QR-Code mit einer Authenticator-App (z.B. Google Authenticator, Authy, 1Password).
4. Geben Sie den 6-stelligen Code zur Bestätigung ein.
5. Speichern Sie die **Backup-Codes** an einem sicheren Ort.

> **Wichtig**: Ohne Backup-Codes verlieren Sie bei Geräteverlust den Zugang zu Ihrem Konto. Speichern Sie die Codes in einem Passwortmanager.

### Deaktivierung

1. Öffnen Sie **Einstellungen → Sicherheit**.
2. Tippen Sie auf **2FA deaktivieren**.
3. Bestätigen Sie mit Ihrem aktuellen 2FA-Code.

## Aktive Sitzungen

Unter **Einstellungen → Sicherheit → Aktive Sitzungen** sehen Sie alle Geräte, auf denen Sie angemeldet sind:

| Information | Beschreibung |
|-------------|-------------|
| Gerätename | Gerätetyp und Betriebssystem |
| Standort | Ungefährer Standort (basierend auf IP) |
| Letzte Aktivität | Zeitpunkt der letzten Nutzung |
| Aktuell | Markierung für das aktuelle Gerät |

### Sitzung beenden

- **Einzelnes Gerät**: Wischen Sie nach links und tippen Sie auf **Abmelden**.
- **Alle anderen Geräte**: Tippen Sie auf **Alle anderen Sitzungen beenden** am Ende der Liste.

## Sprache ändern

Die App-Sprache wird unter **Einstellungen → Allgemein → Sprache** konfiguriert:

- Deutsch (Standard)
- Französisch
- Italienisch
- Englisch

Die Änderung wird sofort übernommen — ein Neustart ist nicht erforderlich.

> **Hinweis**: Die Sprache betrifft nur die mobile App. Die Web-Anwendung hat eine eigene Spracheinstellung unter Einstellungen → Darstellung.

## Passwort ändern

1. Öffnen Sie **Einstellungen → Sicherheit → Passwort ändern**.
2. Geben Sie Ihr aktuelles Passwort ein.
3. Geben Sie das neue Passwort ein und bestätigen Sie es.
4. Tippen Sie auf **Speichern**.

Das Passwort muss mindestens 8 Zeichen lang sein und Gross-/Kleinbuchstaben sowie eine Zahl enthalten.
