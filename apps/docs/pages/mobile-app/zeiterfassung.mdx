# Zeiterfassung

Die mobile Zeiterfassung ermöglicht es, Arbeitszeiten direkt auf dem Smartphone zu erfassen — mit Live-Timer, Kommissionszuweisung und Wochenstatistiken.

## Timer starten

1. Öffnen Sie den Tab **Zeiterfassung** in der Navigation.
2. Tippen Sie auf **Timer starten**.
3. Der Live-Timer zählt in Echtzeit — auch wenn Sie die App minimieren.

> **Hinweis**: Der Timer läuft serverseitig weiter. Sie können die App schliessen und später fortfahren.

## Kommission zuweisen

Vor oder während einer laufenden Erfassung können Sie eine Kommission oder ein Projekt zuweisen:

1. Tippen Sie auf das Feld **Kommission / Projekt**.
2. Wählen Sie aus der Liste oder suchen Sie per Textfeld.
3. Die Zuweisung wird sofort gespeichert.

Ohne Zuweisung wird die Zeit als «Allgemein» erfasst.

## Timer stoppen

Tippen Sie auf **Stopp**. Es erscheint eine Zusammenfassung:

| Feld | Beschreibung |
|------|-------------|
| Startzeit | Wann der Timer gestartet wurde |
| Endzeit | Wann der Timer gestoppt wurde |
| Dauer | Berechnete Arbeitszeit |
| Kommission | Zugewiesenes Projekt (falls vorhanden) |

Bestätigen Sie mit **Speichern** oder verwerfen Sie den Eintrag.

## Wochenstatistik

Unterhalb des Timers sehen Sie ein **Balkendiagramm** mit den Stunden der aktuellen Woche:

- **Grüne Balken**: Erfasste Stunden pro Tag
- **Gestrichelte Linie**: Tagessoll (falls konfiguriert)
- **Gesamtstunden**: Wochensumme am unteren Rand

Wischen Sie nach links, um vergangene Wochen anzuzeigen.

## Manuelle Erfassung

Falls Sie vergessen haben, den Timer zu starten:

1. Tippen Sie auf **+ Manuell erfassen**.
2. Wählen Sie Datum, Start- und Endzeit.
3. Weisen Sie optional eine Kommission zu.
4. Tippen Sie auf **Speichern**.

## Offline-Fähigkeit

Zeiterfassungen funktionieren auch ohne Internetverbindung. Einträge werden lokal gespeichert und bei nächster Verbindung synchronisiert. Mehr dazu unter [Offline-Modus](/mobile-app/offline).
