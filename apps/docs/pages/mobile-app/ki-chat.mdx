# KI-Chat

Der mobile KI-Chat ist Ihr Assistent für schnelle Fragen zu Beständen, Werkzeugen und Bestellungen — direkt auf dem Smartphone.

## Chat öffnen

Tippen Sie in der Navigation auf das **Chat-Symbol** (Sprechblase mit Stern). Der Chat öffnet sich als Vollbild-Ansicht.

## Fragen stellen

Geben Sie Ihre Frage im Textfeld ein und tippen Sie auf **Senden**. Beispiele:

- «Wie viele Kabelbinder sind am Lager Zürich?»
- «Welche Werkzeuge sind überfällig?»
- «Erstelle eine Bestellung für 50 Schrauben M8»
- «Was wurde gestern ausgebucht?»

> **Hinweis**: Der Chat versteht Deutsch und Englisch. Fragen können in natürlicher Sprache gestellt werden.

## Streaming-Antworten

Antworten werden in Echtzeit gestreamt — Sie sehen den Text Wort für Wort erscheinen. Bei datenbasierten Antworten erscheint zunächst ein Lade-Indikator, während die Datenbank abgefragt wird.

## Vorgeschlagene Prompts

Unterhalb des Textfelds erscheinen **Schnell-Prompts** als Chips:

| Prompt | Aktion |
|--------|--------|
| Bestandsübersicht | Zeigt kritische Bestände |
| Überfällige Werkzeuge | Listet nicht zurückgegebene Werkzeuge |
| Letzte Aktivität | Zusammenfassung der letzten Buchungen |
| Bestellung erstellen | Startet einen Bestelldialog |

Tippen Sie auf einen Chip, um die Frage sofort zu senden.

## Aktionen aus dem Chat

Der Assistent kann direkt Aktionen vorschlagen:

- **Bestellung erstellen**: Wird als Entwurf angelegt — Sie bestätigen manuell.
- **Bestand anzeigen**: Link zur Artikeldetailseite.
- **Buchung auslösen**: Bestätigung per Dialog vor der Ausführung.

## Chat-Verlauf

Der Verlauf bleibt innerhalb der Sitzung erhalten. Bei einem App-Neustart beginnt ein neuer Chat. Tippen Sie auf **Neuer Chat** (Besen-Symbol), um den Verlauf manuell zu löschen.

## API-Schlüssel

Der KI-Chat nutzt den in den Web-Einstellungen hinterlegten OpenAI API-Schlüssel Ihrer Organisation. Ohne Schlüssel steht ein eingeschränkter Demo-Modus zur Verfügung.
