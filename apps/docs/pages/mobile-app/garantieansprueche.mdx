# Garantieansprüche

Garantieansprüche können direkt auf dem Smartphone erstellt und verfolgt werden — vom Entwurf bis zur Erledigung.

## Übersicht

Öffnen Sie **Garantieansprüche** über das Menü. Die Liste zeigt alle Ansprüche mit:

- **Claim-Nummer** (z.B. WC-2026-042)
- **Artikel** und Lieferant
- **Status** als farbiges Badge
- **Erstellungsdatum**

## Neuen Anspruch stellen

1. Tippen Sie auf **+ Neuer Anspruch**.
2. Wählen Sie den betroffenen **Artikel** per Suche oder Scanner.
3. Wählen Sie den **Lieferanten** aus der hinterlegten Liste.
4. Beschreiben Sie den **Mangel** im Freitextfeld.
5. Fügen Sie optional **Fotos** als Nachweis hinzu (Kamera oder Galerie).
6. Tippen Sie auf **Absenden**.

> **Hinweis**: Die App prüft automatisch, ob die Garantiefrist des Artikels noch aktiv ist. Falls abgelaufen, erscheint eine Warnung — der Anspruch kann trotzdem eingereicht werden.

Eine eindeutige Claim-Nummer wird automatisch vergeben.

## Status verfolgen

Jeder Anspruch durchläuft folgende Status:

| Status | Beschreibung |
|--------|-------------|
| Entwurf | Noch nicht eingereicht |
| Eingereicht | An den Lieferanten übermittelt |
| In Prüfung | Lieferant bearbeitet den Anspruch |
| Genehmigt | Anspruch akzeptiert |
| Abgelehnt | Anspruch abgelehnt |
| Erledigt | Ersatz, Reparatur oder Erstattung erfolgt |

Tippen Sie auf einen Anspruch, um die Detail-Timeline und alle Kommentare einzusehen.

## Auflösungstypen

Bei genehmigten Ansprüchen wird einer von drei Typen erfasst:

- **Ersatz**: Neuer Artikel wird geliefert
- **Reparatur**: Artikel wird vom Lieferanten repariert
- **Erstattung**: Gutschrift oder Rückzahlung

## Benachrichtigungen

Sie erhalten Push-Benachrichtigungen bei Statusänderungen Ihrer Ansprüche. Die Konfiguration erfolgt unter [Push-Benachrichtigungen](/mobile-app/push-notifications).

## Offline

Entwürfe können offline erstellt werden. Die Einreichung erfolgt automatisch bei nächster Verbindung.
