# Lieferverfolgung

Die mobile Lieferverfolgung zeigt den Status aller Sendungen auf einen Blick — mit Filterfunktion, Timeline und Statusänderung.

## Übersicht

Öffnen Sie **Lieferverfolgung** über die Navigation oder das Menü. Alle aktiven Sendungen erscheinen als Karten mit:

- **Sendungsnummer** und Spediteur-Logo
- **Aktueller Status** als farbiges Badge
- **Voraussichtliches Lieferdatum**
- **Empfänger / Standort**

## Status-Filter

Am oberen Bildschirmrand stehen Filter-Chips zur Verfügung:

| Filter | Beschreibung |
|--------|-------------|
| Alle | Sämtliche Sendungen |
| Bestellt | Bestellung aufgegeben, noch nicht versendet |
| Unterwegs | Sendung beim Spediteur |
| In Zustellung | Letzte Meile, Zustellung heute |
| Zugestellt | Erfolgreich geliefert |
| Überfällig | Lieferdatum überschritten |

Tippen Sie auf einen Chip, um die Liste zu filtern. **Überfällig** ist rot hervorgehoben.

## Timeline

Tippen Sie auf eine Sendung, um die Detail-Timeline zu öffnen:

1. **Bestellung erstellt** — Datum und Benutzer
2. **Versand bestätigt** — Sendungsnummer und Spediteur
3. **Unterwegs** — Letzte Aktualisierung vom Spediteur
4. **Zugestellt** — Lieferdatum und Empfänger

> **Hinweis**: Schweizer Spediteure (Post, DHL, DPD, Planzer, Camion) werden automatisch erkannt. Tippen Sie auf **Tracking öffnen**, um die externe Tracking-Seite des Spediteurs zu öffnen.

## Status ändern

Sie können den Status einer Sendung direkt auf dem Smartphone ändern:

1. Öffnen Sie die Sendungsdetails.
2. Tippen Sie auf **Status ändern**.
3. Wählen Sie den neuen Status aus der Liste.
4. Optional: Fügen Sie eine Notiz hinzu (z.B. «Paket beschädigt»).
5. Bestätigen Sie mit **Speichern**.

Statusänderungen werden sofort synchronisiert und im Aktivitätsprotokoll erfasst.

## Push-Benachrichtigungen

Bei aktivierten Push-Benachrichtigungen erhalten Sie Meldungen bei:

- Versandbestätigung
- Zustellversuch fehlgeschlagen
- Sendung zugestellt
- Sendung überfällig

Benachrichtigungen konfigurieren Sie unter [Push-Benachrichtigungen](/mobile-app/push-notifications).
