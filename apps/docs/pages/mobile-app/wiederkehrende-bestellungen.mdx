# Wiederkehrende Bestellungen

Die mobile Ansicht für wiederkehrende Bestellungen zeigt alle automatischen Nachbestellungen und ermöglicht das schnelle Aktivieren oder Pausieren von Regeln.

## Übersicht

Öffnen Sie **Wiederkehrende Bestellungen** über das Menü. Jede Regel wird als Karte angezeigt:

- **Regelname** (z.B. «Verbrauchsmaterial Lager Bern»)
- **Frequenz**: Täglich, Wöchentlich, Alle 2 Wochen oder Monatlich
- **Nächste Ausführung**: Datum der nächsten automatischen Bestellung
- **Artikelanzahl**: Anzahl der Positionen in der Regel
- **Status-Badge**: Aktiv (grün) oder Pausiert (grau)

## Regeldetails

Tippen Sie auf eine Regel, um die Details einzusehen:

| Feld | Beschreibung |
|------|-------------|
| Lieferant | Zugewiesener Lieferant |
| Frequenz | Bestellintervall |
| Artikelliste | Alle Positionen mit Menge und Einheit |
| Letzte Bestellung | Datum und Status der letzten Ausführung |
| Nächste Bestellung | Geplantes Ausführungsdatum |

## Aktivieren und Pausieren

1. Öffnen Sie die Regeldetails.
2. Tippen Sie auf den **Schalter** neben dem Status.
3. **Aktivieren**: Die Regel wird zum nächsten geplanten Termin ausgeführt.
4. **Pausieren**: Die Regel wird übersprungen, bis sie wieder aktiviert wird.

> **Hinweis**: Beim Pausieren bleiben alle Einstellungen erhalten. Laufende Bestellungen werden nicht abgebrochen.

Alternativ können Sie den Status direkt in der Übersicht ändern, indem Sie auf einer Karte nach links wischen.

## Letzte Ausführungen

Unterhalb der Regeldetails sehen Sie eine **Historie** der letzten Ausführungen:

- **Datum** der Bestellung
- **Status**: Erstellt, Gesendet oder Fehlgeschlagen
- **Bestellnummer** mit Link zur Bestellung

## Verbrauchsanpassung

Falls aktiviert, passt die Regel die Bestellmengen automatisch an den tatsächlichen Verbrauch der letzten 30 Tage an. Ein **Pfeil-Symbol** neben der Menge zeigt an, ob die Menge erhöht oder reduziert wurde.

## Regeln erstellen

Neue Regeln werden in der Web-Anwendung unter **Bestellungen → Wiederkehrend → Neue Regel** erstellt. Die mobile App dient zur Überwachung und Steuerung bestehender Regeln.
