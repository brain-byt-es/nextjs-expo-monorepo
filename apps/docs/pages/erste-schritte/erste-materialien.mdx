# Erste Materialien erfassen

Dieser Abschnitt zeigt Ihnen, wie Sie Ihr erstes Material manuell anlegen. Für grössere Bestände empfehlen wir den [CSV/Excel-Import](/materialien/import).

## Material anlegen

1. Klicken Sie in der linken Navigation auf **Materialien**.
2. Klicken Sie oben rechts auf **+ Material erstellen**.
3. Füllen Sie das Formular aus:

| Feld | Pflichtfeld | Beschreibung |
|------|-------------|--------------|
| Bezeichnung | Ja | Klarer Name, z.B. «Betonschrauben 6×60mm» |
| Artikelnummer | Nein | Interne Nummer aus Ihrem System |
| EAN / Barcode | Nein | 8- oder 13-stelliger EAN-Code |
| Einheit | Ja | Stück, kg, m, L, Palette usw. |
| Kategorie | Nein | Frei wählbare Kategorie |
| Mindestbestand | Nein | Löst Warnung aus wenn Bestand darunter fällt |
| Aktueller Bestand | Nein | Anfangsbestand beim Anlegen |
| Standort | Nein | Welchem Standort ist der Artikel zugeordnet |
| Beschreibung | Nein | Freitext, z.B. technische Details |
| Foto | Nein | Bild des Artikels (JPG, PNG, max. 5 MB) |

4. Klicken Sie auf **Speichern**.

> **Tipp:** Geben Sie möglichst früh einen EAN-Code ein. So können Sie den Artikel später mit der Kamera scannen, ohne ihn manuell suchen zu müssen.

## Kategorien sinnvoll nutzen

Kategorien helfen beim Filtern und Auswerten. Beispiele:

- Baustoffhandel: «Befestigungsmittel», «Dämmmaterial», «Werkzeug», «Verbrauchsmaterial»
- Werkstatt: «Ersatzteile», «Betriebsstoffe», «Schutzmaterial»
- Gastronomie: «Reinigungsmittel», «Einwegmaterial», «Technik»

Kategorien werden automatisch aus bestehenden Einträgen vorgeschlagen, können aber auch frei getippt werden.

## Mindestbestand und Warnungen

Wenn Sie einen **Mindestbestand** definieren, erhalten Sie eine Benachrichtigung sobald der aktuelle Bestand diesen Wert unterschreitet. Voraussetzung: Benachrichtigungen sind in den [Einstellungen](/einstellungen/benachrichtigungen) aktiviert.

Der Status wird in der Materialübersicht farblich angezeigt:
- **Grün**: Bestand über Mindestmenge
- **Orange**: Bestand erreicht Mindestmenge (≤ 110%)
- **Rot**: Bestand unter Mindestmenge

## Artikel bearbeiten

1. Klicken Sie in der Materialübersicht auf den Artikelnamen.
2. Klicken Sie rechts oben auf **Bearbeiten** (Stift-Symbol) oder drücken Sie `E`.
3. Nehmen Sie Ihre Änderungen vor und klicken Sie auf **Speichern**.

## Artikel archivieren

Artikel die nicht mehr benötigt werden können archiviert werden (sie werden nicht gelöscht, bleiben aber in der Historie):

1. Öffnen Sie den Artikel.
2. Klicken Sie auf **Optionen (⋯) → Archivieren**.
3. Bestätigen Sie die Aktion.

Archivierte Artikel werden in der Standardansicht ausgeblendet. Sie können sie jederzeit über den Filter **Archiviert** wieder anzeigen und reaktivieren.
