# Profil & Konto

Verwalten Sie Ihre persönlichen Kontoeinstellungen unter **Einstellungen → Profil & Konto**.

## Persönliche Daten ändern

- **Name**: Angezeigt in der App und auf Lieferscheinen die Sie erstellt haben.
- **E-Mail-Adresse**: Ihre Login-E-Mail. Änderung erfordert Bestätigung der neuen Adresse.
- **Profilbild**: JPG oder PNG, wird in der Navigation und auf Lieferscheinen angezeigt.
- **Sprache**: Deutsch, Französisch, Italienisch oder Englisch (betrifft nur Ihre persönliche Ansicht).

## Passwort ändern

1. Klicken Sie auf **Passwort ändern**.
2. Geben Sie Ihr aktuelles Passwort ein.
3. Geben Sie das neue Passwort ein (min. 8 Zeichen, 1 Grossbuchstabe, 1 Zahl).
4. Bestätigen Sie das neue Passwort.
5. Klicken Sie auf **Passwort speichern**.

## Zwei-Faktor-Authentifizierung (2FA)

Erhöhen Sie die Sicherheit Ihres Kontos mit einem zweiten Faktor:

1. Klicken Sie auf **2FA aktivieren**.
2. Scannen Sie den QR-Code mit einer Authenticator-App (Google Authenticator, Authy, 1Password).
3. Geben Sie den 6-stelligen Code zur Bestätigung ein.
4. Speichern Sie die **Backup-Codes** sicher ab (für den Fall, dass Sie den Authenticator verlieren).

## Aktive Sessions

Unter **Sicherheit → Aktive Sessions** sehen Sie alle Geräte, auf denen Sie aktuell angemeldet sind (Web-Browser, mobile App). Sie können einzelne Sessions oder alle anderen Sessions beenden.

## Konto löschen

Das Löschen Ihres persönlichen Kontos ist unter **Konto löschen** möglich. Beachten Sie:

- Wenn Sie der letzte Administrator einer Organisation sind, muss zuerst ein anderer Administrator ernannt oder die Organisation gelöscht werden.
- Alle von Ihnen erstellten Daten (Lieferscheine, Korrekturen usw.) bleiben in der Organisation erhalten und werden einem neutralen Systembenutzer zugewiesen.
