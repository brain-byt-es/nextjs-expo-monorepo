# Darstellung

Passen Sie das Erscheinungsbild der Web-App an Ihre Präferenzen an.

## Dark Mode / Light Mode

Klicken Sie oben rechts auf das Mond-/Sonnen-Symbol um zwischen hellem und dunklem Modus zu wechseln. Die Einstellung wird pro Benutzer gespeichert. Sie können auch **System** wählen — dann folgt die App automatisch der Betriebssystem-Einstellung.

## Sprache

Verfügbare Sprachen: Deutsch, Français, Italiano, English. Die Spracheinstellung betrifft die gesamte Oberfläche der App (Menüs, Formularbeschriftungen, Fehlermeldungen). Dokumente wie Lieferscheine werden in der Sprache gedruckt, die der Administrator unter **Einstellungen → Branding** festgelegt hat.

## Tabellendichte

Wählen Sie zwischen:
- **Kompakt**: Mehr Zeilen, kleinere Zeilenhöhe — für grosse Bildschirme
- **Standard**: Ausgewogenes Verhältnis
- **Komfortabel**: Grössere Abstände — für Touch-Nutzung

## Startseite anpassen

Wählen Sie, welche Seite nach dem Login angezeigt wird:
- Dashboard (Standard)
- Materialübersicht
- Werkzeugübersicht
- Lieferscheine

## Animationen reduzieren

Für Nutzende mit Bewegungsempfindlichkeit: Aktivieren Sie **Animationen reduzieren** um alle Übergangsanimationen zu deaktivieren.
