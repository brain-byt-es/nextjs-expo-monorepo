# Dashboard anpassen

Das Dashboard ist vollständig anpassbar. Wählen Sie aus 14 Widgets die für Ihre Arbeit relevantesten Kennzahlen aus und ordnen Sie sie nach Ihren Wünschen an.

## Verfügbare Widgets (14)

| Widget | Beschreibung |
|---|---|
| **Bestandsübersicht** | Gesamtbestand in CHF und Anzahl Artikel |
| **Mindestbestand-Alert** | Artikel unter Mindestbestand (Ampel) |
| **Letzte Bewegungen** | Letzte 10 Bestandsbewegungen |
| **Werkzeug-Ausleihe** | Aktuell ausgecheckte Werkzeuge |
| **Überfällige Werkzeuge** | Werkzeuge nach Rückgabedatum |
| **Lieferscheine** | Offene Lieferscheine nach Status |
| **Anomalien** | Letzte Anomalie-Meldungen |
| **Ablaufdaten** | Artikel mit bald ablaufendem Datum |
| **Nachfrageprognose** | Top-5 Artikel mit Stockout-Risiko |
| **Kalibrierungen** | Fällige und überfällige Kalibrierungen |
| **Versicherungen** | Ablaufende Policen und Garantien |
| **Verbrauchsdiagramm** | Balkendiagramm des Monatsverbrauchs |
| **Top-Entnahmen** | Meistverbrauchte Artikel (Woche/Monat) |
| **Sync-Status** | Mobile Sync-Status und Konflikte |

## Dashboard bearbeiten

1. Klicken Sie oben rechts auf dem Dashboard auf **Anpassen** (Stift-Symbol).
2. Das Dashboard wechselt in den **Bearbeitungsmodus**:
   - Widgets werden mit einem Griff-Symbol (**⠿**) versehen
   - Inaktive Widgets erscheinen in der Widget-Seitenleiste

### Widget hinzufügen

1. Ziehen Sie ein Widget aus der Seitenleiste auf das Dashboard.
2. Lassen Sie es an der gewünschten Position los.
3. Das Widget wird sofort mit echten Daten befüllt.

### Widget verschieben

Klicken und halten Sie das Griff-Symbol (**⠿**) eines Widgets und ziehen Sie es an die neue Position. Andere Widgets weichen automatisch aus (automatisches Reflow).

### Widget entfernen

Klicken Sie auf das **×** oben rechts am Widget um es zu entfernen. Die Daten gehen nicht verloren — Sie können das Widget jederzeit wieder hinzufügen.

### Widget-Grösse anpassen

Einige Widgets können in der Grösse angepasst werden:

- Klicken Sie mit Rechtsklick auf das Widget → **Grösse ändern**
- Wählen Sie: Klein (1 Spalte), Mittel (2 Spalten), Gross (3 Spalten, nur auf grossen Bildschirmen)

## Responsives Grid

Das Dashboard verwendet ein **12-Spalten-Grid**:

| Bildschirmbreite | Grid-Spalten | Max. Widgets nebeneinander |
|---|---|---|
| < 768 px (Mobile) | 1 | 1 |
| 768–1 024 px (Tablet) | 2 | 2 |
| > 1 024 px (Desktop) | 3–4 | 3–4 |

## Layout speichern und zurücksetzen

- **Speichern**: Das Layout wird automatisch gespeichert wenn Sie den Bearbeitungsmodus verlassen (**Fertig**-Button).
- **Zurücksetzen**: Klicken Sie im Bearbeitungsmodus auf **Standard wiederherstellen** um das Standardlayout wiederherzustellen.

Das Layout wird pro Benutzer gespeichert — jedes Teammitglied kann sein eigenes Dashboard konfigurieren.
