# Benachrichtigungszentrale

Die Benachrichtigungszentrale ist Ihr persönliches Postfach für alle In-App-Ereignisse. Das Glocken-Symbol oben rechts in der Navigation zeigt immer die Anzahl ungelesener Benachrichtigungen an.

## Benachrichtigungszentrale öffnen

Klicken Sie auf das **Glocken-Symbol** (🔔) in der Navigationsleiste. Ein Dropdown zeigt die neuesten Benachrichtigungen — klicken Sie auf **«Alle anzeigen»** für die vollständige Übersicht.

Der Badge-Counter auf dem Glocken-Symbol zeigt bis zu **99 ungelesene** Benachrichtigungen. Bei mehr als 99 wird **«99+»** angezeigt.

## Benachrichtigungstypen

| Typ | Auslöser |
|-----|----------|
| **Mindestbestand** (`low_stock`) | Lagerbestand fällt unter den konfigurierten Mindestbestand |
| **Wartung fällig** (`maintenance_due`) | Wartungsintervall eines Werkzeugs läuft ab (7 Tage Vorlauf) |
| **Genehmigung erforderlich** (`approval`) | Neue Genehmigungsanfrage wartet auf Ihre Entscheidung |
| **Erwähnung** (`comment_mention`) | Sie wurden in einem Kommentar mit @Name erwähnt |
| **Reservierung** | Reservierungsanfrage wurde bestätigt oder abgelehnt |
| **Importergebnis** | Hintergrundimport wurde abgeschlossen |
| **Anomalie** | KI-Anomalie-Erkennung hat einen ungewöhnlichen Vorgang festgestellt |

## Automatische Aktualisierung

Die Benachrichtigungszentrale prüft alle **30 Sekunden** auf neue Benachrichtigungen. Im Vordergrund-Tab geschieht dies automatisch ohne Seitenaktualisierung. Bei einem Seitenload werden Benachrichtigungen immer sofort geladen.

Auf mobilen Geräten werden Benachrichtigungen zusätzlich als [Push-Benachrichtigungen](/mobile-app/push-notifications) zugestellt.

## Als gelesen markieren

- **Einzelne Benachrichtigung**: Klicken Sie auf die Benachrichtigung, oder hover darüber und klicken Sie das Häkchen-Symbol.
- **Alle als gelesen markieren**: Klicken Sie in der Übersicht auf **«Alle als gelesen markieren»**.

Gelesene Benachrichtigungen werden mit gedämpfter Schrift dargestellt und nach 30 Tagen automatisch gelöscht.

## Benachrichtigungen löschen

- **Einzeln löschen**: Hover über eine Benachrichtigung → Klicken Sie auf das **«×»**-Symbol.
- **Alle löschen**: Klicken Sie auf **«Alle löschen»** (mit Bestätigungsdialog).

> **Hinweis:** Gelöschte Benachrichtigungen können nicht wiederhergestellt werden. Das [Aktivitätsprotokoll](/erweitert/aktivitaetsprotokoll) bleibt davon unberührt.

## Benachrichtigungseinstellungen

Unter **Einstellungen → Benachrichtigungen** können Sie pro Benachrichtigungstyp konfigurieren, ob und wie Sie benachrichtigt werden möchten:

- In-App (immer empfohlen)
- E-Mail (täglich zusammengefasst oder sofort)
- Push-Benachrichtigung (nur Mobile App)

Administratoren können zusätzlich festlegen, welche Rollen welche Benachrichtigungstypen erhalten.
