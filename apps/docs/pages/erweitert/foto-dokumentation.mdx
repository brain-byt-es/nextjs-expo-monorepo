# Foto-Dokumentation

Bei jeder Buchung — Materialeingang, Materialausgang, Werkzeug-Checkout oder -Checkin — können Sie ein Foto als Zustandsnachweis aufnehmen oder hochladen.

## Foto bei einer Buchung erfassen

### Im Browser (Desktop)

1. Öffnen Sie den Buchungsdialog (Einbuchen, Ausbuchen, Checkout, Checkin).
2. Klicken Sie im Abschnitt **Dokumentation** auf **Foto hinzufügen**.
3. Wählen Sie:
   - **Datei hochladen**: Bild von Ihrem Computer auswählen (JPG, PNG, WebP, max. 10 MB)
   - **Kamera**: Falls Ihr Computer eine Webcam hat, können Sie direkt aufnehmen
4. Das Bild wird als Vorschau angezeigt. Mit **X** können Sie es verwerfen.
5. Schliessen Sie die Buchung ab — das Foto wird automatisch mitgespeichert.

### In der Mobile App

1. Starten Sie den Buchungsvorgang (Scan → Aktion wählen).
2. Im Buchungsformular tippen Sie auf **Foto aufnehmen**.
3. Die Gerätekamera öffnet sich. Fotografieren Sie den Artikel.
4. Bestätigen Sie das Bild oder nehmen Sie es erneut auf.
5. Das Foto wird mit der Buchung gespeichert.

## Thumbnails in der Buchungshistorie

In der Buchungshistorie eines Artikels (Tab **Verlauf**) werden Buchungen mit Foto durch ein kleines **Kamera-Symbol** und ein Thumbnail gekennzeichnet. Klicken Sie auf das Thumbnail, um das Foto zu vergrössern.

## Vollbild-Vorschau

- **Desktop**: Klicken Sie auf das Thumbnail in der Buchungshistorie. Das Bild öffnet sich in einem Vollbild-Overlay. Mit `Esc` oder dem X-Button schliessen Sie es.
- **Mobile App**: Tippen Sie auf das Thumbnail. Wischen Sie mit zwei Fingern zum Zoomen.

## Pflichtfotos (Checklisten-Integration)

In [Wartungs- und Übergabe-Checklisten](/werkzeuge/checklisten) können einzelne Punkte als **Pflichtfoto** definiert werden. In diesem Fall ist das Aufnehmen eines Fotos zwingend erforderlich, bevor die Buchung abgeschlossen werden kann.

## Speicherung und Datenschutz

- Fotos werden komprimiert und in einem privaten, organisationsgebundenen Speicher abgelegt.
- Sie sind **nicht öffentlich zugänglich** — nur eingeloggte Benutzer Ihrer Organisation können sie einsehen.
- Fotos werden mit der Buchung dauerhaft gespeichert und können nicht einzeln gelöscht werden. Die gesamte Buchung kann nur von Administratoren gelöscht werden.

## Speicherlimits

| Plan | Foto-Speicher |
|------|--------------|
| Starter | 1 GB |
| Business | 10 GB |
| Enterprise | unbegrenzt |

Den aktuellen Speicherverbrauch sehen Sie unter **Einstellungen → Abonnement**.
