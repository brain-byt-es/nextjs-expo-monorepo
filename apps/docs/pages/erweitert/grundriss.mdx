# Grundriss-Ansicht

Mit der Grundriss-Ansicht laden Sie einen Lagerplan oder Gebäudegrundriss hoch und platzieren Lagerorte direkt darauf. So sehen Sie auf einen Blick, wo sich welche Bestände befinden.

> **Hinweis:** Die Grundriss-Ansicht ist ab dem **Business-Plan** verfügbar.

## Grundriss hochladen

1. Öffnen Sie einen Standort unter **Standorte**.
2. Klicken Sie auf die Registerkarte **Grundriss**.
3. Klicken Sie auf **Grundriss hochladen**.
4. Wählen Sie eine Bilddatei (PNG, JPG oder SVG, max. 20 MB).
5. Das Bild wird skaliert und gespeichert.

> **Tipp:** Verwenden Sie eine möglichst hochauflösende Grafik. SVG-Dateien (z.B. aus AutoCAD oder Illustrator) liefern die schärfste Darstellung bei allen Zoomstufen.

## Lagerort-Marker platzieren

Nachdem der Grundriss hochgeladen wurde:

1. Klicken Sie auf **+ Marker hinzufügen**.
2. Klicken Sie auf die Stelle im Grundriss, an der sich der Lagerort befindet (z.B. Regal A, Zone 3).
3. Wählen Sie den **Lagerort** aus der Dropdown-Liste (verknüpft mit bestehenden Unterstandorten).
4. Optional: Vergeben Sie eine **Beschriftung** (z.B. «Regal A3»).
5. Speichern Sie.

Wiederholen Sie diesen Schritt für alle Lagerorte im Grundriss.

## Bestandsstatus-Overlay

Nach der Marker-Konfiguration zeigt der Grundriss den Bestandsstatus als farbige Markierung:

| Farbe | Bedeutung |
|---|---|
| Grün | Bestand OK |
| Gelb | Unter Mindestbestand |
| Rot | Kein Bestand |

Fahren Sie mit der Maus über einen Marker um die Top-5-Artikel dieses Lagerorts zu sehen. Klicken Sie auf den Marker um die vollständige Lagerortansicht zu öffnen.

## Grundriss teilen

Sie können den Grundriss als schreibgeschützte Ansicht freigeben:

1. Klicken Sie auf **Teilen**.
2. Wählen Sie **Öffentlicher Link** oder **Nur für Teammitglieder**.
3. Kopieren Sie den Link.

Personen mit dem Link sehen den Grundriss im Lesemodus ohne Login-Anforderung (öffentlicher Link).

## Mehrere Grundrisse pro Standort

Für mehrstöckige Gebäude oder grosse Lagerhallen können Sie mehrere Grundrisse pro Standort hinterlegen:

1. Klicken Sie in der Grundriss-Ansicht auf **+ Ebene hinzufügen**.
2. Vergeben Sie einen Namen (z.B. «Erdgeschoss», «1. Obergeschoss»).
3. Laden Sie den Grundriss für diese Ebene hoch.
4. Wechseln Sie zwischen Ebenen über die Tab-Leiste oben.
