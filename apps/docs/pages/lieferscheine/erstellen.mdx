# Lieferschein erstellen

## Neuen Lieferschein erstellen

1. Navigieren Sie zu **Lieferscheine**.
2. Klicken Sie auf **+ Lieferschein erstellen**.
3. Wählen Sie den **Typ** (Eingang, Ausgang, Transfer).
4. Wählen Sie den **Quellstandort** (bei Ausgang) oder **Zielstandort** (bei Eingang).
5. Optional: Fügen Sie eine **Referenz** hinzu (z.B. Bestellnummer, Projektnummer).
6. Fügen Sie Positionen hinzu (manuell oder per Scanner).
7. Klicken Sie auf **Abschliessen** um den Bestand zu aktualisieren.

## Positionen hinzufügen

**Manuell:**

1. Klicken Sie auf **+ Position hinzufügen**.
2. Suchen Sie den Artikel in der Suchleiste (nach Name, Artikelnummer oder EAN).
3. Geben Sie die Menge ein.
4. Optional: Fügen Sie eine Positionsnotiz hinzu.

**Per Scanner:**

1. Klicken Sie auf das **Scanner-Symbol** neben «Position hinzufügen».
2. Scannen Sie den Barcode des Artikels.
3. Geben Sie die Menge ein (Standard: 1).
4. Der Artikel wird direkt zur Position hinzugefügt.
5. Scannen Sie den nächsten Artikel ohne den Dialog zu schliessen.

Mehr dazu unter [Positionen scannen](/lieferscheine/scannen).

## Lieferschein als Vorlage speichern

Wiederkehrende Lieferscheine (z.B. wöchentliche Ausgabe an ein Fahrzeug) können als Vorlage gespeichert werden:

1. Erstellen Sie den Lieferschein mit allen Positionen.
2. Klicken Sie auf **Optionen (⋯) → Als Vorlage speichern**.
3. Vergeben Sie einen Namen.

Beim nächsten Mal wählen Sie **Aus Vorlage erstellen** und passen nur Menge und Datum an.

## Lieferschein bearbeiten

Lieferscheine im Status **Entwurf** können vollständig bearbeitet werden. **Abgeschlossene** Lieferscheine können nur storniert werden (wodurch der Bestand rückgängig gemacht wird).

> **Hinweis:** Bevor Sie einen Lieferschein abschliessen, prüfen Sie alle Positionen sorgfältig. Nachträgliche Korrekturen sind nur durch Stornierung und Neu-Erfassung möglich.

## Empfänger und Unterschrift

Für Ausgabe-Lieferscheine können Sie einen Empfänger erfassen:

- **Mitglied aus dem Team**: Wählen Sie aus dem Dropdown.
- **Externe Person**: Name und E-Mail manuell eingeben.

Optional: **Digitale Unterschrift** auf dem Tablet oder Mobilgerät. Der Empfänger unterschreibt direkt auf dem Bildschirm.
