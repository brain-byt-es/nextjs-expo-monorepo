# Aus- & Einchecken

Das Checkout/Checkin-System verfolgt, wer ein Werkzeug wann ausgeliehen hat und wann es zurückgegeben wurde.

## Werkzeug auschecken

**Via Web-App:**

1. Öffnen Sie das Werkzeug.
2. Klicken Sie auf **Auschecken** (grüner Button).
3. Wählen Sie **Person** (Mitarbeitende aus dem Team) oder **Standort** (Baustelle, Fahrzeug).
4. Setzen Sie optional ein **Rückgabedatum**.
5. Fügen Sie bei Bedarf eine **Notiz** hinzu (z.B. «Für Projekt Zürich Hauptbahnhof»).
6. Klicken Sie auf **Bestätigen**.

**Via Scanner (mobile App):**

1. Öffnen Sie die Scanner-Funktion in der App.
2. Scannen Sie die Etikette des Werkzeugs.
3. Tippen Sie auf **Auschecken** und wählen Sie Person oder Standort.
4. Bestätigen Sie.

> **Tipp:** Beim Auschecken per Scanner können Sie mehrere Werkzeuge hintereinander scannen ohne die Scanner-Ansicht zu verlassen. Ideal für das schnelle Ausrüsten einer Baustelle.

## Werkzeug einchecken

1. Öffnen Sie das Werkzeug (oder scannen Sie die Etikette).
2. Klicken Sie auf **Einchecken**.
3. Optional: Fügen Sie eine Notiz zum Zustand hinzu (z.B. «Batterie leer», «Kleiner Kratzer auf Gehäuse»).
4. Klicken Sie auf **Bestätigen**.

Das Werkzeug ist sofort wieder als «Verfügbar» markiert.

## Rückgabedatum und Erinnerungen

Wenn beim Auschecken ein Rückgabedatum gesetzt wurde:

- Am Tag des Rückgabedatums erhält die ausleihende Person eine Push-Benachrichtigung.
- Am Folgetag wird der Status auf **Überfällig** (orange) gesetzt.
- Manager und Administratoren erhalten täglich eine Zusammenfassung aller überfälligen Werkzeuge.

## Buchungshistorie

Jedes Werkzeug führt eine vollständige Buchungshistorie:

1. Öffnen Sie das Werkzeug.
2. Klicken Sie auf den Tab **Buchungshistorie**.

Die Historie zeigt:
- Datum und Uhrzeit des Auscheckens / Eincheckens
- Person oder Standort
- Dauer der Ausleihe
- Notizen
- Initiierender Benutzer

## Massenbuchungen

Für das Ausrüsten einer Baustelle mit mehreren Werkzeugen:

1. Gehen Sie zu **Werkzeuge**.
2. Aktivieren Sie die Checkboxen aller benötigten Werkzeuge.
3. Klicken Sie auf **Massenaktionen → Auschecken**.
4. Wählen Sie Person oder Standort.
5. Bestätigen Sie — alle Werkzeuge werden gleichzeitig ausgebucht.

## Werkzeug als defekt melden

Falls ein Werkzeug beschädigt zurückgegeben wird:

1. Checken Sie das Werkzeug ein.
2. Klicken Sie auf **Als defekt melden**.
3. Beschreiben Sie den Schaden.
4. Optional: Foto des Schadens hochladen.

Das Werkzeug wird auf Status «Defekt» gesetzt und kann nicht mehr ausgeliehen werden, bis ein Administrator es wieder freigibt oder zur Wartung schickt.
