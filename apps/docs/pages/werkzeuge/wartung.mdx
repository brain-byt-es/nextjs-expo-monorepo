# Wartung & Service

Die Wartungsplanung stellt sicher, dass Werkzeuge regelmässig geprüft, gewartet und kalibriert werden.

## Wartungsintervall festlegen

Beim [Erfassen eines Werkzeugs](/werkzeuge/erstellen) können Sie direkt ein Wartungsintervall definieren. Nachträglich ändern:

1. Öffnen Sie das Werkzeug.
2. Klicken Sie auf **Bearbeiten**.
3. Passen Sie im Abschnitt **Wartung** das Intervall und das nächste Wartungsdatum an.
4. Speichern Sie.

### Intervalltypen

| Typ | Beispiel |
|-----|---------|
| Zeitbasiert (täglich) | Tägliche Sichtprüfung von Sicherheitsausrüstung |
| Zeitbasiert (wöchentlich) | Wöchentlicher Kettenöl-Check bei Kettensägen |
| Zeitbasiert (monatlich) | Monatlicher Service bei Kompressoren |
| Zeitbasiert (jährlich) | Jährliche UVV-Prüfung elektrischer Betriebsmittel |
| Betriebsstundenbasiert | Alle 50 h, 100 h, 500 h |

## Wartung durchführen

1. Öffnen Sie das Werkzeug.
2. Klicken Sie auf **Wartung durchführen**.
3. Füllen Sie das Wartungsprotokoll aus:
   - **Datum** der Wartung
   - **Durchgeführt von**: Mitarbeitende oder externer Dienstleister
   - **Beschreibung** der durchgeführten Arbeiten
   - **Kosten (CHF)**: Optional für Kostenauswertung
   - **Nächste Wartung**: Wird automatisch berechnet, kann überschrieben werden
   - **Dokumente**: Prüfprotokoll, Rechnung hochladen
4. Klicken Sie auf **Wartung abschliessen**.

Das Datum der nächsten Wartung wird automatisch angepasst.

## Wartungskalender

Unter **Werkzeuge → Kalenderansicht** sehen Sie alle geplanten Wartungen:

- **Grün**: Wartung in über 30 Tagen fällig
- **Orange**: Wartung in weniger als 30 Tagen fällig
- **Rot**: Wartung überfällig

Klicken Sie auf einen Eintrag um direkt das Wartungsprotokoll zu öffnen.

## Wartungserinnerungen

Warnungen werden automatisch gesendet:
- **30 Tage vor** der nächsten Wartung: E-Mail-Benachrichtigung an den zuständigen Manager
- **7 Tage vor**: Push-Benachrichtigung an die mobile App
- **Am Tag der Wartung**: Erinnerung in der App

Konfigurieren Sie Empfänger unter [Einstellungen → Benachrichtigungen](/einstellungen/benachrichtigungen).

## Wartungshistorie

Jede durchgeführte Wartung wird dauerhaft gespeichert:

1. Öffnen Sie das Werkzeug.
2. Klicken Sie auf **Wartungshistorie**.

Die Historie zeigt alle Wartungen mit Datum, Durchführenden, Kosten und Notizen. Das vollständige Protokoll kann als PDF exportiert werden — ideal für Audits oder Übergaben an Dritte.

## Werkzeug zur externen Wartung senden

1. Checken Sie das Werkzeug aus und wählen Sie als «Standort»: den Typ **Externer Service**.
2. Tragen Sie den Namen des Dienstleisters ein.
3. Optional: Senden Sie per Klick eine E-Mail an den Dienstleister mit den Werkzeugdaten.

Das Werkzeug zeigt Status «In Wartung» und ist nicht ausleihbar, bis es wieder eingecheckt wird.
