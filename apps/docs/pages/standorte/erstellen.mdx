# Standort erstellen

1. Navigieren Sie zu **Standorte**.
2. Klicken Sie auf **+ Standort erstellen**.
3. Füllen Sie das Formular aus.
4. Klicken Sie auf **Speichern**.

## Felder

- **Name** *(Pflicht)*: Eindeutiger, sprechender Name (z.B. «Lager Zürich-Nord», «Fahrzeug ZH 123 456»)
- **Typ** *(Pflicht)*: Lager, Fahrzeug, Baustelle, Büro oder Sonstiges
- **Adresse**: Strasse, PLZ, Ort, Land — erscheint auf Lieferscheinen
- **Verantwortliche Person**: Mitglied aus dem Team
- **Beschreibung**: Freitext (Öffnungszeiten, Zugangsinformationen usw.)
- **Foto**: Bild des Standorts (Aussenansicht, Lageplan)

## Standort bearbeiten

1. Klicken Sie auf den Standort.
2. Klicken Sie auf **Bearbeiten** (Stift-Symbol).
3. Ändern Sie die gewünschten Felder und speichern Sie.

## Standort archivieren

Standorte können archiviert werden wenn sie nicht mehr aktiv sind (z.B. abgeschlossene Baustelle):

1. Öffnen Sie den Standort.
2. Klicken Sie auf **Optionen (⋯) → Archivieren**.

Alle Bestände des Standorts bleiben in der Datenbank erhalten. Archivierte Standorte erscheinen nicht mehr in Auswahllisten bei Lieferscheinen und Korrekturen.

> **Hinweis:** Ein Standort kann erst archiviert werden wenn keine Werkzeuge mehr dort als «aktueller Standort» geführt sind und der Bestand 0 beträgt. Falls nötig, führen Sie zuerst eine Bestandskorrektur auf 0 durch oder buchen Sie Werkzeuge um.
