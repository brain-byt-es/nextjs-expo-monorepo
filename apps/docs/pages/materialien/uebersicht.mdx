# Materialien – Übersicht

Die Materialübersicht ist der zentrale Ausgangspunkt für die Verwaltung Ihres gesamten Lagerbestands.

## Oberfläche

![Materialübersicht](/screenshots/materialien-uebersicht.png)

Die Übersicht zeigt alle aktiven Materialien in einer Tabelle mit folgenden Spalten:

| Spalte | Beschreibung |
|--------|--------------|
| Foto | Artikelbild (falls vorhanden) |
| Bezeichnung | Name und Artikelnummer |
| EAN | Barcode-Nummer |
| Kategorie | Zugeordnete Kategorie |
| Bestand | Aktueller Lagerbestand mit Einheit |
| Mindestbestand | Definierter Schwellenwert |
| Status | Ampelfarbe (grün/orange/rot) |
| Standort | Primärer Lagerort |
| Zuletzt geändert | Datum der letzten Änderung |

## Suche und Filter

Nutzen Sie die Suchleiste oben oder die Filteroptionen in der Seitenleiste:

- **Suche**: Durchsucht Bezeichnung, Artikelnummer und EAN gleichzeitig. Tastenkürzel: `/` (Schrägstrich)
- **Kategorie**: Mehrfachauswahl möglich
- **Standort**: Zeigt nur Artikel eines bestimmten Standorts
- **Status**: Nur Artikel unter Mindestbestand anzeigen
- **Archiviert**: Archivierte Artikel ein-/ausblenden

> **Tipp:** Kombinieren Sie mehrere Filter gleichzeitig. Die aktiven Filter werden als Tags über der Tabelle angezeigt und können einzeln entfernt werden.

## Ansicht wechseln

Schalten Sie mit den Symbolen oben rechts zwischen **Tabellenansicht** und **Kachelansicht** um. Die Kachelansicht zeigt grössere Bilder und eignet sich besonders für die visuelle Identifikation von Artikeln.

## Spalten anpassen

Klicken Sie auf das Spalten-Symbol oben rechts um zu wählen, welche Spalten angezeigt werden sollen. Ihre Einstellungen werden pro Benutzer gespeichert.

## Sortierung

Klicken Sie auf einen Spaltenheader um aufsteigend zu sortieren, erneuter Klick sortiert absteigend. Die aktive Sortierung wird durch einen Pfeil angezeigt.

## Massenaktionen

Aktivieren Sie die Checkboxen links um mehrere Artikel gleichzeitig zu bearbeiten:

- **Kategorie zuweisen**: Allen ausgewählten Artikeln eine Kategorie zuweisen
- **Standort ändern**: Standort für alle ausgewählten Artikel setzen
- **Archivieren**: Mehrere Artikel gleichzeitig archivieren
- **Exportieren**: Nur ausgewählte Artikel als CSV exportieren

## Tastaturkürzel

| Kürzel | Aktion |
|--------|--------|
| `/` | Suche aktivieren |
| `N` | Neues Material erstellen |
| `I` | Import-Dialog öffnen |
| `E` | Ausgewähltes Material bearbeiten |
| `Esc` | Suche/Filter zurücksetzen |
